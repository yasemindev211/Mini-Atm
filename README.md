# 💳 Mini ATM System

A simple command-line ATM application built with Python. This project simulates basic banking operations such as user authentication, balance management, and money transfers.

## 🚀 Features
-  User registration & login  
-  Deposit and withdraw money  
-  IBAN-based money transfer  
-  Transaction history tracking  
-  Basic financial reporting  
-  Password change functionality  

##  Technologies
- Python  
- CLI (Command Line Interface)
